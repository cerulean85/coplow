- **SOM**: 1년 내 목표 160억 원 (SAM의 5%)
- **경쟁사**: 사람인, 잡코리아, 원티드, 링크드인 KR, 로켓펀치
- **추천 BM**: SaaS 구독 (HR 담당자) + 프리미엄 기능 과금
- **North Star KPI**: 월간 매칭 성공 건수 (채용 확정 기준)

## 섹션 1. 서비스 컨셉 정의

### 핵심 한 줄 정의
> AI가 중소기업 문화와 구직자 커리어 패턴을 분석해, 양쪽 모두 만족하는 채용 매칭을 52일에서 14일로 단축하는 B2B2C 채용 플랫폼

### JTBD (Jobs to be Done)
- **HR 담당자 Job**: "채용 공고를 올린 후 수백 개의 이력서를 일일이 검토하는 시간을 줄이고, 실제로 우리 회사 문화에 맞는 사람을 빠르게 찾고 싶다"
- **구직자 Job**: "스펙 대신 실력과 성향으로 평가받는 중소기업에서 커리어를 성장시키고 싶지만, 어떤 회사가 나에게 맞는지 알기 어렵다"

### 핵심 Pain Point
1. **HR**: 이력서 검토 시간 과다 → 평균 100개 이력서 검토에 40시간 소요 (사람인 2024 HR 리포트)
2. **HR**: 문화 적합성 파악 불가 → 채용 6개월 내 이직률 중소기업 35% (고용노동부 2024)
3. **구직자**: 중소기업 실태 정보 부족 → 복지, 조직 문화, 성장 가능성 파악 방법 없음 (잡코리아 2024: 68%)
4. **구직자**: 스펙 중심 필터링 → 직무 경험·역량 중심이 아닌 학벌/자격증 필터에 걸림

### 가치 제안 (Value Proposition)
- **HR 담당자**: AI 문화 매칭으로 이력서 검토 시간 70% 감소, 채용 소요 기간 52일 → 14일
- **구직자**: 역량 기반 매칭으로 나에게 맞는 중소기업 Top 5 자동 추천, 회사 내부 정보 투명 공개

### USP
1. AI가 기업 문화(직원 리뷰, 조직 패턴, 팀 구성)와 구직자 커리어 패턴을 교차 분석해 **문화 적합도 스코어**를 산출하는 유일한 플랫폼
2. 단순 이력서 매칭이 아닌 **직무 기여도 기반 포트폴리오** 평가로 스펙 외 역량 객관화
3. 채용 확정 전까지 **익명 직원 Q&A** 채널로 회사 실태를 솔직하게 파악 가능

## 섹션 4. User Flow & Information Architecture

### 4.1 핵심 User Flow — HR 담당자: 채용 공고 등록 & 매칭 확인

```
[랜딩페이지] → [무료 트라이얼 신청] → [회사 기본 정보 입력]
    → [채용 포지션 등록] → [AI 문화 분석 실행 (자동)]
    → [매칭 후보 리스트 수신] → [후보 프로필 상세 보기]
    → (관심 여부) → [Yes: 인터뷰 요청 발송] / [No: 다음 후보 검토]
    → [인터뷰 일정 조율] → [채용 확정] → [성공 리포트 수신]
```

### 4.2 핵심 User Flow — 구직자: 프로필 등록 & 매칭 수신

```
[앱 다운로드 / 웹 접속] → [소셜 로그인] → [커리어 정보 입력]
    → [포트폴리오 연동 (GitHub/Notion/직접 업로드)]
    → [문화 성향 테스트 (10분)] → [AI 매칭 분석 (자동)]
    → [추천 기업 Top 5 수신] → [기업 상세 정보 + 익명 Q&A]
    → (지원 여부) → [Yes: 관심 표현 → HR 알림 발송]
    → [인터뷰 일정 수락] → [면접 진행] → [결과 수신]
```

### 4.3 Conditional Flow — 매칭 결과 없음

```
[AI 매칭 분석 완료] → (매칭 기업 수 < 3)
    → [추가 정보 요청 알림] → [프로필 보완 가이드 제공]
    → [프로필 업데이트] → [재매칭 실행]
    → (재매칭 후 결과) → [Yes: 매칭 리스트 제공] / [No: 인재풀 대기 안내]
```

### 4.4 Error Flow — 결제 실패 (HR 구독)

```
[구독 결제 시도] → (결제 실패)
    → [실패 알림 + 원인 안내]
    → [카드 재등록 안내] → [재결제 시도]
    → (성공) → [구독 활성화] / (실패 반복) → [고객센터 연결]
```

### 4.5 Information Architecture (IA)

```
HireMatch
├── HR 대시보드
│   ├── 채용 공고 관리

│   ├── AI 매칭 결과

│   │   └── 후보 상세 프로필

│   │   ├── 일정 조율
│   │   └── 피드백 기록
│   └── 구독 & 결제
│       ├── 플랜 변경
│       └── 청구서 내역
│
├── 구직자 프로필
│   ├── 기본 정보 & 포트폴리오
│   ├── 커리어 성향 테스트
```

## 섹션 6. MVP 정의 & Phase별 로드맵

### MVP 핵심 기능 세트 (Phase 1, 3개월)

1. **소셜 로그인 & 기본 인증** — 진입 장벽 최소화 필수
2. **구직자 프로필 & 포트폴리오 등록** — 매칭 데이터 확보
3. **채용 공고 등록 & 관리** — HR 핵심 워크플로우
4. **AI 문화 매칭 엔진 (v1)** — 핵심 차별 가치 검증
5. **매칭 결과 리스트 제공** — 양측에 결과 전달

> MVP 제외 이유 (Should/Could-have): 익명 Q&A, 성향 테스트, 리포트는 핵심 가치 검증 후 추가

### MVP 검증 지표 & 성공 기준

| 지표 | 측정 방법 | 목표 | Go 기준 |
|-----|---------|------|--------|
| HR DAU | 앱 로그 | 500명/일 (3개월 차) | 400명 이상 |
| 구직자 DAU | 앱 로그 | 2,000명/일 (3개월 차) | 1,500명 이상 |
| 월간 매칭 성공 건수 | 채용 확정 이벤트 | 200건/월 | 100건 이상 |

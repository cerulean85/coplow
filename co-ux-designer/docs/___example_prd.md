# 샘플 PRD: 독서 기록 앱 "BookLog"

> **용도**: UX 설계자 Agent 테스트 및 데모용 샘플 PRD
> **버전**: PRD v1.0
> **작성일**: 2026-04-13
> **작성자**: 서비스 기획자 Agent (샘플)

---

## 1. 서비스 개요

**서비스명**: BookLog
**한 줄 설명**: 독서 습관을 기록하고 관리하는 개인 독서 다이어리 앱
**플랫폼**: iOS / Android 모바일 앱
**MVP 출시 목표**: 2026년 Q3

## 3. 타겟 사용자 페르소나

### 페르소나 A — 주 타겟 (Kim Soyeon, 독서 습관 형성 중인 직장인)
- **나이/직업**: 28세, IT 스타트업 기획자
- **독서 패턴**: 월 2~4권, 주로 출퇴근 시간이나 주말에 읽음
- **목표**: 읽은 책을 기록하고, 독서 목표를 달성하고 싶음
- **Pain Point**:
  - 읽은 책 목록이 흩어져 있어 나중에 찾기 어려움
  - 독서 노트를 쓰다가 귀찮아서 중단하는 경우가 많음
  - 어떤 책을 읽었는지 친구에게 공유하고 싶지만 번거로움
- **사용 환경**: iPhone, 출퇴근 지하철, 주말 카페

### 페르소나 B — 부 타겟 (Park Junho, 독서 마니아)
- **나이/직업**: 42세, 중학교 국어 교사
- **독서 패턴**: 월 8~10권, 장르 가리지 않고 폭넓게 읽음
- **목표**: 연간 독서 통계를 보며 성취감을 느끼고 싶음
- **Pain Point**:
  - 독서 이력이 너무 많아 관리가 어려움
  - 특정 책에 대한 감상을 나중에 다시 보고 싶은데 기억이 안 남

## 4. 사용자 핵심 목표 (User Goals)

| 기능 | 사용자 목표 |
|---|---|
| 책 등록 | "내가 읽은 책을 빠르게 기록하고 싶다" |
| 독서 메모 | "책의 인상적인 구절과 내 생각을 바로 기록하고 싶다" |
| 독서 목표 설정 | "올해 몇 권 읽겠다는 목표를 세우고 진행 상황을 보고 싶다" |
| 독서 통계 | "내 독서 패턴과 성취를 한눈에 보고 싶다" |
| 책 검색 | "읽고 싶은 책을 빠르게 찾아 위시리스트에 추가하고 싶다" |

## 5. 기능 명세 (MVP 기준)

### F-01. 책 등록 및 관리
- **설명**: ISBN 스캔 또는 제목/저자 검색으로 책을 등록하고 읽기 상태를 관리
- **세부 기능**:
  - 바코드(ISBN) 스캔 등록
  - 도서 검색 API 연동 (카카오 책 검색 API)
  - 읽기 상태: 읽고 싶다 / 읽는 중 / 읽었다
  - 독서 시작일, 완독일 입력
  - 별점(5점 만점) 평가
- **우선순위**: MVP

### F-02. 독서 메모(하이라이트 & 노트)
- **설명**: 책을 읽으며 인상적인 구절 또는 생각을 기록
- **세부 기능**:
  - 구절 입력 + 페이지 번호 입력
  - 개인 메모/감상 텍스트 입력
  - 메모에 태그(#감동, #지식, #공감 등) 추가
  - 메모 목록 조회 및 검색
- **우선순위**: MVP

### F-03. 독서 목표 설정
- **설명**: 연간/월간 독서 목표 권수 설정 및 달성 현황 표시
- **세부 기능**:
  - 연간 목표 권수 설정 (기본값: 12권)
  - 현재 진행률(%) 및 달성 예측 표시
  - 목표 달성 시 배지(Badge) 획득
- **우선순위**: MVP

### F-04. 독서 통계 대시보드
- **설명**: 독서 이력 기반 통계 시각화
- **세부 기능**:
  - 이번 달 / 올해 완독 권수
  - 장르별 독서 비율 (파이 차트)
  - 월별 독서량 추이 (바 차트)
  - 평균 완독 기간
- **우선순위**: MVP

### F-05. 책 검색 & 위시리스트
- **설명**: 읽고 싶은 책 탐색 및 위시리스트 관리
- **세부 기능**:
  - 도서 검색 (제목/저자/ISBN)
  - 위시리스트(읽고 싶다) 추가/삭제
  - 위시리스트 목록 조회
- **우선순위**: MVP

### F-06. 소셜 공유 (Post-MVP)
- **설명**: 완독한 책 또는 인상적인 구절을 SNS에 공유
- **세부 기능**:
  - 완독 카드 이미지 생성 (책 표지 + 별점 + 한줄평)
  - Instagram/KakaoTalk 공유
- **우선순위**: Post-MVP

## 6. 우선순위 및 MVP 범위

### MVP (v1.0 출시)
- F-01 책 등록 및 관리
- F-02 독서 메모
- F-03 독서 목표 설정
- F-04 독서 통계 대시보드
- F-05 책 검색 & 위시리스트

### Post-MVP (v1.1 이후)
- F-06 소셜 공유
- 친구 독서 현황 보기
- 독서 클럽/모임 기능
- 프리미엄 플랜 (상세 통계, 무제한 메모)

---

## 7. 비기능 요구사항

- **반응 속도**: 책 검색 결과 2초 이내 응답
- **오프라인 지원**: 메모 작성은 오프라인에서도 가능 (로컬 저장 후 동기화)
- **접근성**: iOS/Android 기본 접근성 API 준수
- **개인정보**: 독서 데이터는 사용자 계정에 안전하게 저장, 제3자 공유 없음

## 📌 업데이트 규칙

- 파일 추가·수정·삭제 시 반드시 이 파일에 기록한다.
- 변경 유형: `Added` (추가) / `Changed` (수정) / `Deprecated` (비권장) / `Removed` (삭제) / `Fixed` (수정)
- 버전 형식: `vMAJOR.MINOR` — 구조 변경 시 MAJOR 증가, 내용 수정 시 MINOR 증가
- 가장 최신 버전이 최상단에 위치한다.
